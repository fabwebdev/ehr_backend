Healthcare Charts Backend API

Welcome to the Healthcare Charts Backend API. This Express.js application provides RESTful endpoints for managing healthcare data.

Note: This is a backend API service. All endpoints are prefixed with /api.

Available Endpoints

GET /api/patients
Retrieve a list of all patients
GET /api/patients/:id
Retrieve a specific patient by ID
POST /api/patients
Create a new patient
PUT /api/patients/:id
Update an existing patient
DELETE /api/patients/:id
Delete a patient
GET /api/pain-assessments
Retrieve a list of all pain assessments
GET /api/pain-assessments/:id
Retrieve a specific pain assessment by ID
POST /api/pain-assessments
Create a new pain assessment
GET /api/nursing-notes
Retrieve a list of all nursing clinical notes
GET /api/nursing-notes/:id
Retrieve a specific nursing note by ID
POST /api/nursing-notes
Create a new nursing clinical note
Authentication: Most endpoints require authentication via JWT tokens. Include the token in the Authorization header as "Bearer [token]".

For detailed API documentation, please refer to the project documentation or contact the development team.
